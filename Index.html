Вход на сайт

Для доступа к материалам из локальной сети колледжа авторизация не требуется. Учащимся заочной формы обучения для авторизации требуется ввести шифр из зачетной книжки (<номер группы>/<номер по списку>) в поле «Имя пользователя» и свою фамилию (с большой буквы) в поле «Пароль»

Электронная библиотека

Ресурс для ознакомления с материалами в электронном виде. Получить копию электронного материала возможно из внутренней сети колледжа, либо пройдя процедуру авторизации на ресурсе (возможно только для учащихся заочной формы получения образования).
